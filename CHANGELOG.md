# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-15

### Added
- Interactive Leaflet.js map with earthquake markers
- Tectonic lineaments overlay from Myanmar Tectonic Map 2011
- Statistics panel showing total events, max/avg magnitude, and avg depth
- Magnitude distribution bar chart
- Magnitude filter slider
- Layer toggle controls for earthquakes and fault lines
- Multiple basemap options (Dark, Satellite, Terrain, Light)
- Color-coded magnitude legend
- Major events list (M ≥ 5.0) with click-to-zoom
- Events timeline bar chart
- Magnitude over time scatter chart
- Responsive design for different screen sizes
- Dark theme with modern aesthetics
- Custom styled popups for earthquake details

### Data
- Earthquake data from USGS (March 28, 2025 – January 15, 2026)
- 181 earthquake events along the Sagaing Fault
- Includes M 7.7 Mandalay earthquake and M 6.7 aftershock

### Technical
- Pure HTML/CSS/JavaScript implementation (no framework dependencies)
- Leaflet.js for mapping
- Chart.js for data visualization
- CSS custom properties for theming
- Responsive grid layout
